# Wow Murloc Landing Page

Welcome to the Wow Murloc project! This is a private World of Warcraft server designed for an immersive gaming experience.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The main HTML document for the landing page.
- **css/style.css**: Contains styles for the landing page, including layout and responsiveness.
- **js/main.js**: JavaScript functionality for the landing page.
- **assets/logo.svg**: The logo for the Wow Murloc project.
- **assets/icon.svg**: The icon for the Wow Murloc project.
- **credits/CONTRIBUTORS.md**: Acknowledgment of the creators and contributors of the project.
- **.gitignore**: Specifies files and directories to be ignored by version control.
- **LICENSE**: Licensing information for the project.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd wow-murloc-landing
   ```
3. Open `index.html` in your web browser to view the landing page.

## Usage

Feel free to explore the landing page and learn more about the Wow Murloc project. You can also contribute to the project by following the guidelines in the `CONTRIBUTORS.md` file.
